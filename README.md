## What is LaunchCart?

LaunchCart is a full-stack e-commerce starter application designed to accelerate development of online product catalogues with customer inquiry workflows. It provides a modern React frontend, a robust ASP.NET Core backend, and PostgreSQL persistence—ready for customization and deployment.
